# nvim-api-client 

A lightweight, blazing-fast API client built entirely in Lua for Neovim. Stop context-switching between your editor and external tools like Postman or Insomnia. Test your endpoints exactly where you write your code.

---

##  Features

* **Full Method Support**: Seamlessly execute `GET`, `POST`, `PUT`, and `DELETE` requests.
* **JSON Integration**: Send complex JSON payloads directly from your buffers.
* **Live Response View**: View formatted API responses (status codes, headers, and body) in a dedicated, syntax-highlighted split window.
* **Minimalist Design**: Built to be fast and stay out of your way, leveraging `curl` under the hood.
* **Buffer-Driven**: Works with `.http` or `.rest` files for a clean, documentation-like workflow.
